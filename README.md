# Spring AI MCP Knowledge Base

A production-ready MCP (Model Context Protocol) server that gives Claude Code
semantic search access to Spring AI documentation.

## Project Layout

```
springai-mcp/
├── README.md
├── requirements.txt
├── .mcp.json                  # Claude Code MCP config
│
├── data/
│   └── docs/                  # Put your .md files here
│
├── store/                     # Generated by ingest.py (git-ignored)
│   ├── springai.index         # FAISS binary index
│   └── metadata.json          # Chunk metadata
│
├── ingest.py                  # One-time ingestion script
├── server.py                  # FastAPI retrieval server
└── mcp_server.py              # MCP stdio server (called by Claude Code)
```

## Quick Start

```bash
# 1. Create & activate venv
python3 -m venv .venv
source .venv/bin/activate

# 2. Install deps
pip install -r requirements.txt

# 3. Copy your .md files
cp ~/Documents/springai-kb/*.md data/docs/

# 4. Ingest (builds FAISS index)
python ingest.py

# 5. Start retrieval API
uvicorn server:app --host 0.0.0.0 --port 8000 --reload

# 6. (Optional) Test the MCP server directly
echo '{"jsonrpc":"2.0","id":1,"method":"tools/list"}' | python mcp_server.py
```

## Claude Code Config

Copy `.mcp.json` to your project root or `~/.claude/` — Claude Code will
auto-discover the MCP server.
